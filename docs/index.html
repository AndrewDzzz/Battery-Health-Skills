Quick Commands

SOH Feature + Uncertainty Demo

Synthetic fleet-like telemetry → integrity → training + uncertainty

python soh-field-demo/scripts/run_battery_field_demo.py \
  --generate-data \
  --n-cells 6 \
  --cycles-per-cell 14 \
  --points-per-cycle 120 \
  --out-dir ./runs/ev-fleet-demo

Feature extraction

Derive interpretable features from cycle telemetry for modeling.

python soh-modeling-upgrade/scripts/extract_soh_features.py \
  --input telemetry.csv \
  --output soh_features.csv \
  --timestamp-col timestamp \
  --cycle-col cycle_id \
  --current-col current \
  --voltage-col voltage \
  --temp-col temperature \
  --soc-col soc \
  --soh-proxy-col capacity

Security + Integrity

Run anti-tamper checks before any model score is trusted.

python battery-telemetry-integrity/scripts/check_telemetry_integrity.py \
  --input telemetry.csv \
  --time-col timestamp \
  --id-col cell_id \
  --voltage-col voltage \
  --current-col current \
  --temp-col temperature \
  --output-dir runs/telemetry

Standard workflow

  1. 01Ingest telemetry and run integrity gates.
  2. 02Extract SOH and mechanism-aware features.
  3. 03Train model with uncertainty diagnostics.
  4. 04Run domain shift checks when moving between fleets or cells.
  5. 05Publish risk and rollout artifacts.
